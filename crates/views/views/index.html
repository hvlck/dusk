About

What is an ID?

An ID is the text you use to identify your shortened link. For instance, if you have an ID meaning-of-life, then the generated link will be whatever-site.tld/link/meaning-of-life.

What is a stop code?

A stop code allows you to delete a link. Go to the delete page to do so.

What is this?

Dusk is a URL shortener - it generates shorter links which can be used to link to longer URLs.
